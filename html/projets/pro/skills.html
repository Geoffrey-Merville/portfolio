Nom : Dawan Skills

Description du projet : Dawan Skills est une application web de quizzes permettant aux alternants de l'entreprise de s'évaluer sur leurs compétences.

Contexte de Réalisation et Missions : L'entreprise est un organisme de formation proposant divers cursus diplômants, et il est nécesssaire de pouvoir évaluer les compétences de chacun des alternants dans ce but.
Dawan Skills vient répondre à ce besoin en proposant des quizzes sur divers thèmes de sorte de pouvoir évaluer au mieux les compétences de chacun.

Une première version du site étant déjà en production, ma mission consistait à développer de nouvelles fonctionnalités pour l'application en collaboration avec le reste de l'équipe.
Outre le développement de nouvelles fonctionnalités, je devais également participer au débogage de l'application et à la refonte du code de celle-ci afin d'améliorer la maintenance et les performances du site.
J'ai également dû participer à la rédaction de la JSDoc et des tests unitaires et d'intégration de l'application car ceux-ci étaient soit manquants, soit invalides.

Le développement se tenait au sein d'une équipe d'autres alternants et notre méthode de travail reposait sur une méthode agile.

Version et état du projet : Le projet est disponible en production ici.

Stack Technique : Conception Modèle Objet HTML 5 SCSS AngularJS 14 TypeScript Java 8 Spring Boot 2 Maven 3 JPA Hibernate MariaDB Git + Gitlab Refactoring Tests Unitaires
