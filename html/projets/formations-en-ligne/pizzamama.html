Nom : Pizza Mama

Description du projet : Pizza Mama est un mini projet HTML réalisé à but éducatif. Il représente le menu en ligne d'une hypothétique pizzeria nommée "Pizza Mama".

Contexte de Réalisation : version sans back-end (HTML only) | Ce mini projet HTML a été réalisé en tant qu'exercice dans le cadre d'une formation en ligne sur Python. Il s'incrit dans la partie dédiée sur l'apprentissage rapide des langages HTML et CSS de la formation avant de l'intégrer à un back-end Django. Voir ici pour la version avec back-end Django.

Version et état du projet : Le projet est disponible en production ici

Stack Technique : HTML 5 CSS 3
